How To? - top

How to Select layers in PSD?

You can use Move Tool (V) for fast layer selection.



How to edit each section in PSD?

Each section is separated with each folder with differ color marks.



How to change images in PSD?

Place the image in image placeholder layer with clipping mask. So fit the image related position within the box.



How to edit text in PSD?

Select the text layer and change the text.
